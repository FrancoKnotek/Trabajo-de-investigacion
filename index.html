¿Cómo se integra JavaScript a HTML?.

Hay varias formas de hacer que un script se integre a HTML, una de esas formas es la propia etiqueta SCRIPT que se coloca entre el head y el body,
haciendo esto; podrás pegar el codigo dentro de la etiqueta para que este se imprima en la pagina.
Otro metodo es utilizando la misma etiqueta SCRIPT, pero con el atributo SR; pudiendo importar archivos externos al HTML
