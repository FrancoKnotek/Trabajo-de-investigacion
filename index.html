7.¿ Qué es el objeto dom en JS ? Y¿ Cómo trabaja ? .

El objeto DOM o Document Object Model, es una interfaz multiplataforma que trata un documento XML o HTML como una estructura de árbol.Esta funciona de forma tal que facilita una representación estructurada y define de que manera los programas pueden acceder, al fin de modificar.estructura, contenido, etc.

el Object Map tiene la funcion de transformar los elementos de una lista, devolviendo una nueva lista con los elementos ya transformados
